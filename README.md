# 給啊娜的信

**給啊娜的信** adalah website statis berupa surat dan ucapan ulang tahun untuk
Marshanda Fadillah yang siap dipublikasikan melalui GitHub Pages.

## Menjalankan secara lokal

Buka `index.html` melalui browser. Untuk hasil yang paling konsisten, jalankan
server lokal sederhana dari folder ini.

## Mempublikasikan melalui GitHub Pages

1. Buat repository publik baru di GitHub.
2. Unggah seluruh isi folder ini ke bagian paling atas repository.
3. Buka **Settings → Pages**.
4. Pada **Build and deployment**, pilih **Deploy from a branch**.
5. Pilih branch utama, folder `/ (root)`, lalu tekan **Save**.

Alamat website akan berbentuk:
`https://NAMA-PENGGUNA.github.io/NAMA-REPOSITORY/`
